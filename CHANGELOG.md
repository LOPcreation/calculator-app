# Changelog

## Version 1.0.0 - Initial Release

### Features
- ✅ Full-featured scientific calculator
- ✅ 6-category unit converter (Length, Weight, Temperature, Volume, Area, Time)
- ✅ Real-time currency converter with 30+ currencies
- ✅ Material Design 3 UI with dark/light theme support
- ✅ Offline support for calculator and unit converter
- ✅ 1-hour exchange rate caching
- ✅ Fallback mock data for currency conversion

### Technical Highlights
- Kotlin + Jetpack Compose
- MVVM Architecture with StateFlow
- OkHttp3 + Gson for API integration
- Kotlin Coroutines for async operations
- ProGuard code shrinking (~4-5 MB APK)

### Known Issues
None
